## Calculator for assembling a computer

## Install
```bash
git clone https://github.com/Enchu/assembly.git

cd assembly

npm run dev
```

# How to use
```bash
touch .env

echo "NEXT_PUBLIC_DB_URL=your_api_key_here" >> .env
echo "NEXT_PUBLIC_DB_NAME=your_db_name_here" >> .env
```
